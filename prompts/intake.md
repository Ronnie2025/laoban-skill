# 老板 Skill 录入 Prompt

## 任务

你将接收用户提供的老板相关材料，并为后续生成 `history memory`、`persona.md`、`work.md` 做录入整理。

## 你需要拿到的信息

### 1. 基础定位

- 这个老板是什么角色
- 他更偏业务、产品、项目、技术、管理，还是混合
- 用户最想提炼他的哪类能力

### 2. 材料来源

按来源分类：

- 录音转写
- 会议纪要
- 聊天记录
- 文档 / 长文
- 用户主观描述

### 3. 目标输出

确认用户更看重哪些部分：

- 追问方式
- 方案评审方式
- 项目推进方式
- 风险闭环方式
- 表达语气
- 对人方式

### 4. 更新模式

- 本次是首次创建，还是增量更新
- 是否已有 `persona.md` / `work.md` / `history memory`

## 输出格式

```markdown
## 角色定位
- ...

## 材料清单
- [来源类型] 文件名 / 片段说明

## 提炼重点
- ...

## 当前模式
- 首次创建 / 增量更新
```
